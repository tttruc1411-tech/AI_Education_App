# 💡 Hint: Rotate Image

## Goal
Rotate the image by 90 degrees.

## Main Hint
Use the `rotate_image()` function with parameter `angle`.

**Common rotation angles:**
- `90` = Rotate 90 degrees clockwise
- `180` = Rotate 180 degrees (upside down)
- `270` = Rotate 270 degrees (or -90 degrees)

## Code Structure
```python
# 1. Initialize camera
# 2. In the loop:
#    - Get frame
#    - Rotate image: rotated_frame = rotate_image(camera_frame=..., angle=90)
#    - Display rotated image
# 3. Close camera
```

## Required Function:
- `rotate_image(camera_frame=..., angle=...)` - Rotate image
