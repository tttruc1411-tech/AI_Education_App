## 🎯 Cách sử dụng

1. Khi người dùng chọn một lesson, ứng dụng sẽ load step 1
2. File tutorial tương ứng sẽ được hiển thị trong `StepRequirementsPanel`
3. Người dùng đọc yêu cầu và viết code
4. Sau khi hoàn thành, chuyển sang step tiếp theo
5. Sau khi hoàn thành tất cả steps, mở khóa challenge

## 📝 Format của mỗi file

Mỗi file tutorial bao gồm:
- **Mục tiêu**: Mô tả ngắn gọn mục đích của step
- **Các bước thực hiện**: Hướng dẫn chi tiết từng bước
- **Gợi ý**: Tips và tricks hữu ích
- **Kết quả mong đợi**: Checklist để tự kiểm tra
