# Lesson 3 - Step 4: Color Variations
## Task
Experiment with different colors.

## Instructions
1. Initialize the camera
2. In the loop, get a frame
3. Draw 4 circles with different colors: green, red, blue, yellow
4. Each circle at a different position
5. Display the image
6. Close the camera

## Required Functions
- `Init_Camera()` - Initialize camera
- `Get_Camera_Frame(capture_camera)` - Get frame
- `Draw_Circle(camera_frame, center_x, center_y, radius, color)` - Draw circle
- `Show_Image(camera_frame)` - Display image
- `Close_Camera(capture_camera)` - Close camera

## Hints
- Available colors: "green", "red", "blue", "yellow", "white"
- Place 4 circles at 4 corners of the screen
- Example positions: (100, 100), (540, 100), (100, 380), (540, 380)
