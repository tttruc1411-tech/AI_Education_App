# Lesson 1 - Step 1: My First Camera
## Task
Initialize the camera and display live video.

## Instructions
1. Use `Init_Camera()` to initialize the camera
2. Inside the `while True` loop, use `Get_Camera_Frame()` to get a frame
3. Use `Show_Image()` to display the frame
4. Finally, use `Close_Camera()` to close the camera

## Required Functions
- `Init_Camera()` - Initialize the camera
- `Get_Camera_Frame(capture_camera)` - Get a frame from the camera
- `Show_Image(camera_frame)` - Display the frame
- `Close_Camera(capture_camera)` - Close the camera

## Hints
- Save the result of `Init_Camera()` to the `capture_camera` variable
- The `while True` loop will run continuously to display video
- Remember to pass the correct parameters to each function
