# Lesson 3 - Step 3: Multiple Shapes
## Task
Draw multiple different shapes on the same image.

## Instructions
1. Initialize the camera
2. In the loop, get a frame
3. Draw a green rectangle
4. Draw a red circle
5. Draw another yellow rectangle
6. Display the image with all shapes
7. Close the camera

## Required Functions
- `Init_Camera()` - Initialize camera
- `Get_Camera_Frame(capture_camera)` - Get frame
- `Draw_Rectangle(camera_frame, x, y, width, height, color)` - Draw rectangle
- `Draw_Circle(camera_frame, center_x, center_y, radius, color)` - Draw circle
- `Show_Image(camera_frame)` - Display image
- `Close_Camera(capture_camera)` - Close camera

## Hints
- Draw each shape one by one on the same frame
- Choose positions so shapes don't overlap
